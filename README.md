PLC기기, 센서 연결 필요. PLC기기 미연결시 정상작동 안 됨.
에너지 절약형 스마트팩토리 미니 모델 구축

기술
-Languages-
Python (주요 학습/평가/데이터 처리 전부 Python 기반)

C++ (아두이노IDE에서 ESP32 제어 로직 작성)

SQL (Oracle DB 및 CSV 처리 후 SQL 기반 관리/조회)

Ladder Logic (PLC) (XG5000, XBC-DR14E 활용 제어 로직 작성)

-Frameworks & Libraries-
Flask, Flask-SocketIO (에너지 전환/IDS 대시보드 웹 구현, 실시간 송출)

paho-MQTT (ESP32/라즈베리파이 ↔ 서버 통신)

cx_Oracle (Oracle DB 연동)

-Databases-

OracleDB (MES/ERP 로그 기록, IDS 결과 저장)

-IoT Platforms & PLC-

ESP32 (조명/센서 제어, MQTT 연동)

Mosquitto MQTT (브로커 운영, IoT 장치 ↔ 서버 통신)

LS ELECTRIC XBC-DR14E(PLC(Programmable Logic Controller))

-Tools-

Oracle SQL Developer (Oracle DB 관리)

XG5000, XP-Builder (PLC 래더 프로그램 작성)

-Development Environments-

Visual Studio Code (Flask 서버 실행, 자동화 환경)
